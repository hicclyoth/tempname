Lesson 1: Overview of C# and Configuring the Environment

Introduction to C# and its development tools, including Visual Studio and .NET.

Introduction

C# is a modern object-oriented programming language created by Microsoft. It runs on the .NET framework and is used to develop various applications such as games, web apps, and desktop apps. C# is strongly typed, which means each variable must have a defined type. It supports object-oriented programming (OOP) and can be used across platforms with .NET Core and .NET 5+. A common use case for C# is game development with Unity. Development tools for C# include Visual Studio (VS), a complete integrated development environment (IDE) for coding, debugging, and testing, and Visual Studio Code (VS Code), a lightweight editor that supports C# plugins.
