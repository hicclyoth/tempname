Lesson 2: Data Types, Variables, and Constants

Learn about C#'s basic data types, variables, and immutable constants using const.

Introduction

In C#, variables store data that can change while the program is running, whereas constants hold data that remains the same throughout the program's lifecycle. C# provides a variety of data types to represent different kinds of data. Some basic data types include int for whole numbers, double for decimal values, char for individual characters, string for text, and bool for true/false values.
